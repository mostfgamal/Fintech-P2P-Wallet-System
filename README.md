#  P2P Digital Wallet System
**Role:** Senior Technical Business Analyst / Product Owner  
**Domain:** Fintech (Payments & Transfers)

##  Project Overview
Designed the core architecture and functional specifications for a Peer-to-Peer (P2P) wallet system. The system allows users to perform real-time money transfers, bill payments, and wallet top-ups via credit cards.

###  My Contribution
* **Requirements Engineering:** Authored detailed SRS and User Stories.
* **System Design:** Created Sequence Diagrams for transaction flows using .
* **API Design:** Defined RESTful API contracts (JSON) for Mobile-to-Backend communication.
* **Compliance:** Ensured workflows adhere to **ISO 8583** standards and PCI-DSS security guidelines.
